# gbizinfo-cli

## 概要

gBizINFO（法人情報API）を操作するためのCLIツール。経済産業省が提供する法人基本情報・財務情報・補助金受給履歴等を検索・取得する。

## 対象API

- **提供元**: 経済産業省
- **API**: gBizINFO REST API (https://info.gbiz.go.jp/api/index.html)
- **認証**: 一部機能はAPIキー不要、詳細情報はアカウント登録（無料）が必要
- **形式**: REST/JSON

## 主な機能

- 法人情報の検索（法人名、所在地、法人番号など）
- 法人詳細情報の取得（基本情報、財務概要、届出・認定情報）
- 補助金受給履歴の取得
- 調達情報（官公庁入札・契約実績）の取得
- 特許情報の取得
- 職場情報（従業員数、女性比率等）の取得

## 既存ツールの状況

- Ruby gem（1つ）とCLI（1スター）のみ
- 主要言語向けの成熟したツールは存在しない

## 開発方針

- Go言語で実装
- サブコマンド構成（`gbizinfo search`, `gbizinfo get`, `gbizinfo finance` など）
- 出力形式: JSON（デフォルト）、テーブル表示
- 法人デューデリジェンス・コンプライアンスチェックのワークフロー対応
- Claude Code等のAIエージェントからの利用を想定した設計
- 日本語ドキュメント・ヘルプメッセージ
